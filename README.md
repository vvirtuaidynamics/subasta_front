# Subasta App (subasta-app)

Proyecto básico de quasar2 con vite.

## Install the dependencies

```bash
yarn
# or
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
npm run dev
```

### Construir app para producción

```bash
npm run build
```
